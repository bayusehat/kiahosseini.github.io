Yesterday my colleague asked me to fix her HTC Desire 500.
The problem was related to Android booting that got stuck in fastboot screen.
I wanted to give a shot to solve it. Good experience to share with you :)

Unfortunately, none of the options in boot screen worked, even Reset Factory.
So I decided to install a Stock ROM.
I found {% include link.html href="http://forum.xda-developers.com/desire-500/development/rom-htc-desire-500-android-4-1-2-t2849467" external=true text="this topic" %} very helpful to solve the problem.

First, I tried to use Windows 10 to solve the problem.
But Windows could not recognize the device!
Forget it! It is easier to install android tools in Ubuntu (16.04) and no need to recognize the device in a hard way :)

Install android tools in Ubuntu. Run the commands below:

{% include code.html file_name="install_android_tools.sh" language="bash" %}

Download {% include link.html href="http://j.gs/5648760/1344011-103" external=true text="ROM" %} and {% include link.html href="http://j.gs/5648760/firmware-1344011" external=true text="firmware" %}.
Copy the ROM*.zip in your SD Card.

We need to flash the recovery.

For more detailed information visit {% include link.html href="http://www.xda-developers.com/how-to-install-clockworkmod/" external=true text="here" %}.

Connect your device with USB cable to your PC.
You should see the fastboot usb title on your screen.
Download {% include link.html href="https://docs.google.com/file/d/0B6WBFlAKqe30c0kwdG1VZC1FTlU/edit" external=true text="CWM-Recovery_PhilZ_by_SPaWn_500.rar" %}
Extract CWMRecovery_D500_Philz.img.

Before we start installing the recovery image, we need to unlock the bootloader.
So, go and sign up in {% include link.html href="https://www.htcdev.com/" external=true text="htcdev.com" %} and follow the instruction there, to get your device's Unlock_code.bin.

Then try to Install zip from the SD Card (ROM*.zip).
